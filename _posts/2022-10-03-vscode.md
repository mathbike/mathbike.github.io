| - | - |
| `code` | run / open blank |

| <kbd>Alt</kbd>+<kbd>x</kbd> | run python file |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>a</kbd> | comment highlighted |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>l</kbd> | select all occurences |
