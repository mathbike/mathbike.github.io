## Set up the Python environment

```sh
python3 -m venv .venv
echo ".venv

instance/

.pytest_cache/
.coverage
htmlcov/

dist/

*.egg-info/" > .gitignore

```

```terminal
pip install Flask
pip freeze > requirements.txt
```
