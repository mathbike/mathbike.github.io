## Set up the Python environment in vscode

```sh
APP_NAME="flask_test"
mkdir $APP_NAME
python3 -m venv .venv
echo ".venv

*.pyc
__pycache__/

instance/

.pytest_cache/
.coverage
htmlcov/

dist/
build/
*.egg-info/" > .gitignore
mkdir .vscode
echo "{ \"python.terminal.activateEnvironment\": true }" > .vscode/settings.json
source .venv/bin/activate
pip install Flask
pip freeze > requirements.txt
```
