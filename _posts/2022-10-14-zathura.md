Install:
```terminal
sudo pacman -S zathura
```

Commands:

| Command | Description |
| - | - |
| <kbd>Ctrl</kbd>+<kbd>r</kbd> | Invert colors |
| <kbd>s</kbd> | Fit width |
| <kbd>a</kbd> | Fit page |
| <kbd>d</kbd> | Side by side |
| <kbd>-</kbd>or<kbd>+</kbd> (<kbd>Fn</kbd>) | Zoom |
