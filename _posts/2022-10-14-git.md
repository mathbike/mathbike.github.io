- Install:
```terminal
sudo pacman -S git
```

---

- Initialize a repository:
```terminal
git init
```








# clone a repository
git clone <repository>
# staging changes
git add <path>
# add all
git add .
# remove changes
git reset <path>
# status
git status
# commit
git commit
